# Project Overview
  This project analyzes customer purchasing behavior during the Diwali festival using Power BI. The dashboard helps understand which products, customer groups, and   locations generated the highest sales during the festive season.
## Objective
  - To analyze Diwali sales performance
  - To identify top-selling products and categories
  - To understand customer demographics and buying behavior
  - To help businesses make better sales and marketing decisions
## Tools Used
  - Power BI
  - Microsoft Excel (for data cleaning and preparation)
  - Dax formulas
## Dataset Information
  The dataset includes information such as:
  - User_ID
  - Cust_name
  - Product_ID
  - Gender
  - Age Group  
  - Age
  - Marital_Status
  - State
  - Zone
  - Occupation
  - Product_Category
  - Orders
  - Amount
## Key Insights
  - Total sales and total orders analysis
  - Sales comparison by gender and age group
  - Sales analysis by state and city
  - Top product categories and products
  - Customer purchasing patterns during Diwali
## Dashboard Features
  - Identified which age group spent the most
  - Found top states contributing to sales
  - Highlighted most popular product categories
  - Compared male vs female purchasing behavior
## Conclusion
   The dashboard helps businesses understand customer trends during Diwali and supports data-driven decisions for marketing, inventory planning, and sales 
   strategies.
